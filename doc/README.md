# Documentació
Llistat d'alguns dels punts que han de quedar explicats en aquesta carpeta. Poden ser tots en aquest fitxer o en diversos fitxers enllaçats.

És obligatori modificar aquest document!!

## Documentació bàsica MÍNIMA
 * Objectius
 * Arquitectura bàsica
   * Tecnologies utilitzades
   * CSS, HTML, PHP, Java i MySQL
   * Interrelació entre els diversos components
   * El css s'enllaca amb l'HTML i el JS tambe s'enllaca amb l'HTML. La base de dades s'enllaca amb el PHP amb una llibreria o plugin MYSQLDO
 * Com crees l'entorn de desenvolupament
 * El creare utilitzant el VSCODE, despres utilitzant el IntelljIDEA. Per fer servir la base de dades utilitzarem el adminer i el phpMyAdmin.
 * Com desplegues l'aplicació a producció
 * Llistat d'endpoints de l'API de backend (també podeu documentar-ho amb swagger)
    * Rutes
   * Exemples de JSON de peticó
   * Exemples de JSON de resposta i els seus codis d'estat 200? 404?
 * Aplicació Android
